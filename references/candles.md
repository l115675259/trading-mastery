# 日本蜡烛图技术体系 —— 从入门到精通

> 基于史蒂夫·尼森《日本蜡烛图技术新解》整理，用最通俗的话，把整套知识体系讲透。

---

## 全书总览

这本书讲了**五大武器**：

| 武器 | 作用 | 好比 |
|------|------|------|
| 蜡烛图 | 看多空力量消长，抓反转信号 | 显微镜——看细节 |
| 差异指数 | 判断超买超卖，看趋势方向 | 温度计——量体温 |
| 三线反向突破图 | 判断大趋势，过滤噪音 | 望远镜——看大局 |
| 砖形图 | 最简单的趋势跟踪 | 傻瓜相机——简单直接 |
| 钥匙图 | 最灵活的日本新价图 | 瑞士军刀——功能最多 |

**核心理念**：蜡烛图是基本功，新价图用来定大势，差异指数用来测极端状态。五者结合，才是完整的日本技术分析体系。

# 第一部分：蜡烛图

## 第一章 蜡烛图是什么，为什么用它

### 1.1 一句话讲清楚

蜡烛图就是把每天（或每小时、每周）的开盘价、收盘价、最高价、最低价，画成一根蜡烛的形状。

- 蜡烛的**身体**（实体）：开盘到收盘
- 蜡烛的**上下引线**（影线）：最高和最低

### 1.2 为什么比条形图好

| 蜡烛图能做到的 | 条形图做不到的 |
|----------------|----------------|
| 一眼看出今天多头赢还是空头赢 | 要仔细算才知道 |
| 长上影线告诉你"高价被拒绝" | 看不出这个心理含义 |
| 1~3天就能发出反转信号 | 经常要几周才看得出来 |
| 颜色直观反映市场情绪 | 只有黑白线段 |

### 1.3 蜡烛图的历史

17世纪，日本稻米期货市场诞生。一位叫本间宗久的交易员发现：**稻米的价值和价格之间存在差异，而这个差异来自市场情绪**。他由此开创了研究市场情绪来预测价格的道路。

蜡烛图的核心奥义，在《黄金泉——三猿金钱录》里有三句话：

1. **"不见邪"**：看见大牛市时，别追进去，反而要想着卖。阴线和阳线会不断轮回，牛市里酝酿着熊市。
2. **"不听邪"**：听到利好消息别马上冲进去，先看看市场怎么反应。重要的不是消息本身，是市场对消息的反应。
3. **"不言邪"**：研究好自己的计划就执行，别到处问别人意见，越问越乱。

## 第二章 蜡烛图的基础语言

### 2.1 实体：多空力量最直接的表达

**长白实体**（收盘远高于开盘）：多头完全控制局面。
- 出现在低价区 → 可能是底部的第一个信号
- 从支撑位反弹 → 确认支撑有效
- 突破阻力位 → 信号非常强烈
- **最重要的功能：长白实体本身构成支撑位** → 支撑区间为「实体中点」到「下影线底端」。价格回抽到这个区域，是买点。

**长黑实体**（收盘远低于开盘）：空头完全控制局面。
- 出现在高价区 → 可能是顶部的第一个信号
- 从阻力位砸下来 → 确认阻力有效
- 砸破支撑位 → 空头力量非常强
- **最重要的功能：长黑实体本身构成阻力位** → 阻力区间为「实体中点」到「上影线顶点」。价格反弹到这个区域，是卖点。

> 🔑 **判断长实体的标准**：实体长度至少是之前蜡烛线实体的 **3倍**。

**纺锤线**（实体非常小）：多空在拉锯，谁也干不过谁。
- 这意味着之前的趋势**正在失去力量**。
- 纺锤线 + 大成交量 = 警惕！
  - 低价区放量纺锤线 → 筹码积聚（有人暗中吃货）
  - 高价区放量纺锤线 → 筹码派发（聪明钱在出货）

**十字星**（实体为零，开盘价=收盘价）：市场进入"犹豫状态"，像站在十字路口。
- 十字星出现后，行情大概率要转向。
- 十字星的上影线顶点 = 阻力位，下影线底端 = 支撑位。
- **十字星之后，等一根比十字星高的收盘价才能做多；等一根比十字星低的收盘价才能做空。**

**十字星的特殊变体：墓碑十字星**
开盘价、收盘价、最低价全部集中在同一点，上影线很长。这是最熊的十字星形态——把墓碑竖起来了。出现在高位时意味着多头"死得很惨"。实体下方完全没有影线，意味着多方的反抗为零。

### 2.1+ 开盘价的前瞻信号

不等收盘也能获得信息：
- **开盘在前一根白实体中点以下** → 空头信号，激进者可准备卖出
- **开盘在前一根黑实体中点以上** → 多头信号，激进者可准备买入

这个方法在股票市场比期货市场更有效，因为期货开盘容易大幅跳空，而股票开盘价通常接近前一天收盘，一旦偏离就更值得注意。

### 2.2 影线：市场拒绝的价格

**长上影线**：市场曾经冲到高位，但被狠狠打了回来。
- 在高价区出现 → 卖压沉重，多头疲惫。
- 意味着"这个高价，市场不接受"。

**长下影线**：市场曾经砸到低位，但被强势拉起。
- 在低价区出现 → 买盘强劲，空头撤退。
- 意味着"这个低价，有人抢着要"。

**高浪线**（上下影线都长 + 实体很小）：市场完全失去方向感，多空打架打成平手。出现在趋势后，趋势可能要变。

## 第三章 蜡烛图组合形态 —— 核心交易信号

> 本章介绍的所有形态，都可以用「合成蜡烛线」来判断其多空性质。方法是：取这个形态第一根蜡烛的开盘价、全部蜡烛中最高的高点、最低的低点、最后一根蜡烛的收盘价，画成一根蜡烛。合成蜡烛上影线越长 / 实体越小 / 实体在底部 → 偏空；下影线越长 / 实体在顶部 → 偏多。碰到非标准形态时，画一根合成蜡烛看看——这是日本文献里最推荐的校验手段。

### 3.1 单根蜡烛线反转形态

#### 锤子线（底部反转，看涨）

四个条件缺一不可：
1. **必须出现在一段明显的下跌之后**
2. 实体很小（黑白都可以），在价格区间的上端
3. 下影线至少是实体的 **3倍** 以上
4. 上影线很短或没有

**怎么操作**：激进的在锤子线当天就少量买入；稳健的等回抽确认支撑后再买。止损设在锤子线下影线最低点下方。

> 锤子线 + 弹簧线（价格一度跌破支撑又弹回来） = 双重确认，信号更强。

#### 吊颈线（顶部反转，看跌）

形态和锤子线一模一样，但**出现在上涨趋势之后**。

**关键点：必须等第二天的确认！** 第二天的收盘价要低于吊颈线的实体，吊颈线的空头信号才算成立。如果第二天反而继续涨，吊颈线信号作废。

#### 流星线（顶部反转，看跌）

- 上影线很长，实体很小（黑白都可以），实体在价格区间的下端。
- **必须出现在上涨趋势之后**。
- 长长的上影线表示"多头冲上去，被空头打了回来"。

> 流星线 + 上插线（价格一度突破阻力位又跌回来） = 强烈的空头信号。

---

### 3.2 双蜡烛线形态

#### 乌云盖顶（顶部反转，看跌）

- 第一根是长白实体（上升动能很强）
- 第二根黑线**高开低走**，收盘切入第一根白线实体的**中点以下**

**强度判断**：黑线切入白线越深，信号越强。乌云盖顶出现后，形态的最高点构成未来的阻力位。

> 股票市场宽容一点：第二根开盘只需高于第一天收盘价就成。期货市场严格：第二根开盘必须高于第一天最高价。

#### 刺穿形态（底部反转，看涨）

乌云盖顶的反面：
- 第一根是长黑实体
- 第二根白线**低开高走**，收盘切入第一根黑线实体的**中点以上**

白线切入黑线越深，信号越强。

#### 吞噬形态

**空头吞噬**（顶部反转，看跌）：一根长黑实体完全吞掉前面一根小白实体。
**多头吞噬**（底部反转，看涨）：一根长白实体完全吞掉前面一根小黑实体。

> 吞噬形态的信号强度超过乌云盖顶和刺穿形态，因为它"连窝端了"。

**一个容易被忽略的细节**：如果吞噬形态中两根蜡烛线实体差不多长，说明多空均衡，行情可能走成横盘而不是反转。

**吞噬形态的两级严格度**：
- 第一级（基本）：后一根实体包裹前一根实体即可
- 第二级（严格）：后一根的高低点也包裹前一根的上下影线，信号更强

> 股票市场宽容规则：后一根的开盘价只要等于前一根的收盘价，也视为有效吞噬（因为股票开盘通常在昨收附近）。

#### 最后吞噬顶 / 最后吞噬底

- **最后吞噬顶**：形态和多头吞噬一样（长白实体包小黑实体），但它出现在**上涨趋势之后**——这是见顶信号。日本人说这是"双双殉情"，你爱上市场的那一刻，就是你陪市场一起死的时候。
- **最后吞噬底**：形态和空头吞噬一样（长黑实体包小白实体），但它出现在**下跌趋势之后**——这是见底信号。

#### 包孕形态（Harami）

长实体之后紧跟着一个小实体，小实体完全被长实体包在里面。它表达的意思是：**市场的动能正在衰竭**。

**颜色组合影响可靠性**：
- 下跌趋势中，第一根为白实体 → 更偏多（长白本身就有多头含义）
- 上升趋势中，第一根为黑实体 → 更偏空（长黑本身就有空头含义）

**变体**：
- **包孕十字**（小实体变成十字星）→ 信号更强
- **高价包孕**（小实体靠近大实体上端）→ 大概率变成横盘
- **低价包孕**（小实体靠近大实体下端）→ 大概率变成横盘

---

### 3.3 窗口（缺口）

**窗口 = 西方技术分析中的"跳空缺口"**，但在蜡烛图里含义更丰富。

**核心原则**：
1. 上升窗口 = 上涨趋势继续 = **支撑位**
2. 下降窗口 = 下跌趋势继续 = **阻力位**
3. **"反向走势遇窗口而止"**：上升后的回调会在窗口位置获得支撑；下跌后的反弹会在窗口位置遇阻。

**最重要的规则**：
- 如果收盘价**封住了上升窗口**（跌破窗口底部），意味着上涨趋势结束。
- 如果收盘价**封住了下降窗口**（涨回窗口顶部上方），意味着下跌趋势结束。

**三窗口理论**：连续出现三个同方向的窗口，说明趋势走到了极限，"物极必反"，该反转了。

**操作要点**：
- 窗口开启后，等 **3个交易日**不封闭，说明窗口的支撑/阻力有效。
- 盘中曾封闭窗口但收盘又拉回来 → 不算真封闭，趋势还在。
- 反转信号 + 同方向的窗口 → 双倍确认。

**双黑跳空**：下降窗口之后紧跟着两根黑线 → 非常熊的信号。

**流星线与上升窗口冲突怎么判**：流星线本身是空头信号，但如果它同时开启了一个上升窗口，则窗口的多头含义压过流星线——**单个形态不能孤立看**，流星线的空头信号要等收盘跌到窗口下方才成立。

**跳空十字星**：下跌过程中出现一个跳空向下的十字星 → 卖压增强。

---

### 3.4 三根或更多蜡烛线形态

#### 黄昏星（顶部反转，非常可靠）

三根线：
1. 一根长白实体（牛市还在跑）
2. 一根小实体或十字星（与第一根之间有跳空，实体不接触）
3. 一根长黑实体（深度切入第一根白实体）

这就像一个人跑步：第一步还在全力冲刺，第二步突然停下来了，第三步直接往回跑。转折非常明显。

#### 启明星（底部反转）

黄昏星的反面：
1. 一根长黑实体（熊市在延续）
2. 一根小实体或不接触实体的十字星
3. 一根长白实体（强势上攻）

> 第二根如果是十字星，叫「启明十字星」；如果这个十字星与前后两根都有跳空缺口，叫「弃婴形态」——最罕见的底部形态，信号极强。

#### 崩盘十字星（顶部反转的极端版）

和黄昏星类似，但关键差别：十字星是**向下跳空**出现在长白实体下方的，之后再来一根向下跳空的黑线。这是"大跌的前兆"。

---

### 3.5 迭创新高（低）

如果市场出现**8~10根连续（或几乎连续）的新高或新低**（中间可以插一两根横盘蜡烛线），说明市场已经到了"过热"或"过冷"的极端状态，反转的概率大增。

日本交易员把这种形态称为"酒田图形身体里的骨头"，可见其重要性。

> 计算规则：先找到前一段走势的最低点（上升趋势中）或最高点（下降趋势中），以此为准往后数。中间可以穿插1~3根横向蜡烛，但如果出现剧烈反向走势（大跌或大涨），计数器清零重新数。

## 第四章 蜡烛图不是孤岛 —— 整体技术面的五大原则

> 第一遍阅读时记住：**蜡烛图信号 ≠ 交易指令**。信号只是必要条件，后面四条原则才决定做不做。

### 4.1 止损：承认错误才能活下来

- 见乌云盖顶做空？止损点设在乌云盖顶最高点上方。
- 见锤子线做多？止损点设在锤子线最低点下方。
- 见窗口支撑做多？止损点设在窗口底部下方。

当价格突破你的止损位时，不要幻想，不要一厢情愿。"一厢情愿的成本总是太高昂，它就像倚靠白云的梯子——只会导致坠落。"

### 4.2 风险与收益的权衡：不值得就不做

一个蜡烛图信号出现，不等于就要下单。**先算账**：
- 入场价到止损价 = 你的风险（准备亏多少钱）
- 入场价到目标价 = 你的收益（可能赚多少钱）

**只有当收益远大于风险时，才值得出手。** 日本人说："勇士的潜能是一张拉满的弓，关键在于放弦的时机。"

> ⚠️ 蜡烛图的固有局限：**不提供目标价位**。确定目标位要借助西方工具——趋势线、斐波那契回调、前支撑/阻力位等。进场前如果算不出目标价，就先别进。

### 4.3 趋势：顺风跑永远比逆风跑快

核心策略：
- **上升趋势中**：看到多头蜡烛信号就买；看到空头信号就平多仓（不反手做空）。
- **下降趋势中**：看到空头蜡烛信号就卖；看到多头信号就平空仓（不反手做多）。

**"形态的位置比形态本身更重要。"** 一个支撑位上的刺穿形态，和一个随便位置上的空头吞噬形态相比，前者更可能构成有效的反转。

### 4.4 做市场变色龙：情况变了就认

你判断某个阻力位不能被突破，但价格偏偏收盘往上穿了。这时候不要死撑，马上改变判断。交易就是不断假设、验证、修正的过程。比剑决斗中反应慢的人会死，交易中反应慢的人会亏光。

### 4.5 计算机和蜡烛图：别让程序替你做主观判断

- 蜡烛图形态识别有**主观成分**，不完全是数学问题。
- 一个"长得像但不完全标准"的形态，只要位置对（在重要支撑/阻力位、在超买/超卖区），就应该当成标准形态对待。
- 计算机只能帮你筛选，不能帮你决策。

### 4.6 实体群组合——肉眼扫盘术

在一段横盘整理中，直接数：黑实体多还是白实体多？黑的更长还是白的更长？如果一组蜡烛里黑线数量多且长度大→空头占优，横盘后向下突破概率大。不用软件，肉眼一秒就能判断。

### 4.7 "该涨不涨"法则

一个利多消息出来，价格如果连前期高点都突破不了，甚至不涨反跌——最强的卖出信号。反过来也一样：利空消息出来价格不跌→该跌不跌，反向看多。永远看市场对消息的反应，不要看消息本身。

# 第二部分：差异指数

## 第五章 差异指数：市场温度计

### 5.1 什么是差异指数

```
差异指数 = (收盘价 - 移动平均线) ÷ 移动平均线 × 100%
```

比如当前价格是 110，25日均线是 100。差异指数 = +10%。意思是：**价格比均线高了10%**。

> 偏离指数是同一个东西的另一种写法：偏离指数 = 收盘价 ÷ 移动平均线 × 100%。110÷100=110%，和差异指数的+10%完全等价。

### 5.2 三大用法

**用法一：判断超买超卖**

每个品种都有自己的极端值。比如某只股票，13周差异指数到了 +10% 就是超买，到了 -10% 就是超卖。你需要自己去测试你的品种的极端值。
- 超买区 + 空头蜡烛信号 → 双倍确认，卖出
- 超卖区 + 多头蜡烛信号 → 双倍确认，买入

> 日本人常用均线周期：短期 **5日、9日、25日**；中长期 **13周、26周、75日、200日**。短期均线上穿长期 = **黄金交叉**（看多）；下穿 = **死亡交叉**（看空）。

**用法二：判断趋势方向**

差异指数往上走 → 当下处于上涨趋势。差异指数往下走 → 当下处于下跌趋势。在超买和超卖之间的中间地带，用这个来判断方向。

**用法三：检测背离**

价格创了新高，但差异指数没能同步创出新高 → 空头背离，趋势即将反转。价格创了新低，但差异指数没能同步创出新低 → 多头背离，反弹在即。

> 极端差异指数只说明市场在"狂热"或"恐慌"中，**不保证一定反转**——它只告诉你"继续沿原方向走的概率正在降低"。必须等蜡烛图或其他工具的信号来确认。

# 第三部分：新价图（三种日本独门图形）

## 第六章 三线反向突破图

### 6.1 这是什么

一种只画收盘价、忽略时间的图表。白线 = 价格在涨，黑线 = 价格在跌。只有**创出新高或新低**时才画线。名字里的"三线"意思是：当连续出现三根白线或三根黑线后，必须**反向突破这三根线的最低点或最高点**，才能画一根反转线（转向线）。

### 6.2 核心交易规则

- 见白线 = 看涨信号
- 见黑线 = 看跌信号
- 连续三根白线 = 确认上升趋势
- 连续三根黑线 = 确认下降趋势
- 黑色转向线（突破三根白线最低点）= 趋势由涨转跌
- 白色转向线（突破三根黑线最高点）= 趋势由跌转涨

**等待进一步确认**：有些交易者不在转向线出现时立即动手，而等转向线后**再出现一根同方向的线**（比如白转向线后再来一根白线）。多等一步，出错率更低，但进场也晚了一步。

### 6.3 著名日本交易口诀

**做多口诀**：「在黑鞋上面有白套装，白套装上面伸出脖颈时买入。」

解读：
- 黑鞋 = 一根小黑线（卖压衰减）
- 白套装 = 白色转向线（反转向上）
- 脖颈 = 转向线后的一根小白线（确认信号）

**做空口诀**：「在脖颈下面有黑套装，黑套装下面出现黑鞋时卖出。」

解读：
- 脖颈 = 上升末端的一根小白线（上涨乏力）
- 黑套装 = 黑色转向线（反转向下）
- 黑鞋 = 转向线后的一根小黑线（确认信号）

### 6.4 迭创新高（低）在三线反向突破图上的应用

与蜡烛图一样：8~10根连续白线→市场超买，有回调压力；8~10根连续黑线→市场超卖，有反弹动力。

### 6.5 三线反向突破图上的趋势线与支撑阻力

蜡烛图/条形图中的趋势线、双顶、支撑/阻力线，直接搬到三线反向突破图上一样有效。而且由于三线反向突破图极简，支撑和阻力位更加一目了然。

### 6.6 敏感度可调

把反转标准从"三线"改成"二线"→ 信号更频繁，适合短线。改成"四线"或"五线"→ 信号更少，更适合长线抓大行情。

## 第七章 砖形图

### 7.1 这是什么

最简单的新价图。每块"砖头"都一样高。价格涨够一个固定幅度，就加一块白砖；价格跌够一个固定幅度，就加一块黑砖。涨跌幅度不到标准，图上什么也不会出现。

### 7.2 核心交易规则

**规则极其简单**：
- 白砖出现 = 买入信号
- 黑砖出现 = 卖出信号

优点：简单直接，在明确趋势中能让你牢牢抓住行情的主段。缺点：横向盘整时会导致频繁进出、反复止损。

## 第八章 钥匙图（Kagi图）

### 8.1 这是什么

钥匙图是最流行的日本新价图。线的粗细随价格变化：价格突破前高，线变粗（阳线）；价格跌破前低，线变细（阴线）。线之间的短横线叫做"拐点线"，连接反向的线段。

### 8.2 核心交易规则

- **见阳线买入（线变粗时）**
- **见阴线卖出（线变细时）**

### 8.3 肩与腰（高低点循环）

- 前高 = **肩**（Shoulder）
- 前低 = **腰**（Waist）

如果肩和肩比节节升高，腰和腰比节节上移 → **强势市场**，做多。如果肩和腰都节节走低 → **弱势市场**，做空。

一旦"肩越来越高、腰越来越高"的循环被打破（比如新腰低于前腰），就要警惕趋势可能生变。

### 8.4 多级突破

有些交易者不满足于突破一个前高或前低，会等二级突破（突破两个前高/前低）甚至三级突破，才确认趋势反转。等得越久，概率越高，但获利空间越小。

### 8.4+ 极性转换原则在钥匙图上的应用

和蜡烛图一样，钥匙图上曾经的阻力位被突破后会变成支撑位，曾被跌破的支撑位会变成阻力位。钥匙图的优势——每一根线的端点非常明确，支撑/阻力一目了然。

### 8.5 阴阳线长度的比较

- 阳线（粗线） > 阴线（细线） → 多头占优
- 阴线 > 阳线 → 空头占优
- 阴阳线等长 → 像蜡烛图的十字星，多空均衡

### 8.6 中点法则（非常重要）

- 上涨后发生回调，但回调低点没有触及**前面那根长阳线的中点** → 市场仍然强势。如果此后越过前高，买入信号。
- 下跌后发生反弹，但反弹高点没有触及**前面那根长阴线的中点** → 市场仍然弱势。如果此后跌破前低，卖出信号。

> 如果一连串反弹都被卡在某一根长阴线的中点下方，说明多头反攻极其虚弱——反复攻一个位置却始终拿不下，最终大概率向下破位。同理，反复下探同一根长阳线中点却始终砸不穿，向上突破的概率大增。

### 8.7 百分比钥匙图

与其设定一个固定转向值（比如跌3元就转向），不如用百分比（比如3%）。好处是：股价从50元涨到100元后，3%就自动从1.5元变成3元，跟着价格走。**3%是日本短线交易者的常用值，5%适合中长线。**

### 8.8 双窗口形态（钥匙图独有的反转形态）

**双窗底**（看涨）：
- 先有一个低点（腰 W1）
- 中间的反弹高点（肩 S1）低于 W1
- 再创出的新低点（腰 W2）高于 S1
- 这种"腰高、肩低"的结构就像两个窗口打开了

**双窗顶**（看跌）：
- 先有一个高点（肩 S1）
- 中间的回调低点（腰 W1）高于 S1
- 再创出的新高点（肩 S2）低于 W1

### 8.9 三佛顶与三佛底

就是西方技术分析里的"头肩顶"和"头肩底"，日本人叫"三佛顶"和"三佛底"。右肩被突破时，形态确认完成。

**三佛形态的强度判断**：
- 右佛的反弹被压在前一根长阴线中点以下→多头反攻虚弱，空头信号更强
- 右佛的回调被前一根长阳线中点以上托住→空头无力深入，多头信号更可靠
- 如果实现二级突破（同时突破两个肩或腰）→ 最可靠的确认

### 8.10 镊子顶（双顶）

钥匙图上两个等高或接近的高点，日本人称为"镊子顶"——像一把镊子夹住了价格。这是强阻力信号。注意：钥匙图上的双顶确认通常比蜡烛图晚好几根线，这是趋势工具的固有滞后性——比蜡烛图慢，但比蜡烛图稳。

### 8.11 钥匙图与蜡烛图的信号速度差异

**钥匙图的反转信号总是比蜡烛图慢几拍。** 蜡烛图可能在3月初就见到顶部形态，而钥匙图要到4月才确认。这不是缺陷——钥匙图的设计目标就是过滤假信号、只抓大趋势。用钥匙图定方向，用蜡烛图找时机，这才是正确的分工。

### 8.12 迭创新高（低）在钥匙图上的应用

与蜡烛图一样：如果钥匙图上出现连续8~10个肩部新高（或腰部新低），说明市场走到极端，反转的概率大增。

## 总结：五大工具怎么搭配用

| 工具 | 用它干什么 | 什么时候用 |
|------|------------|------------|
| 蜡烛图 | 找具体的买卖信号 | 每天都在看，精细操作 |
| 差异指数 | 判断超买超卖、看背离 | 确认蜡烛信号的可靠性 |
| 三线反向突破图 | 判断中期大盘方向 | 顺势而为，定多空基调 |
| 砖形图 | 最简单的趋势跟踪 | 不想动脑时用，做大趋势 |
| 钥匙图 | 灵活的趋势判断 + 多角度辅助 | 功能最多，适配不同风格 |

**实战组合思路**：

1. 先用**三线反向突破图**（或钥匙图）判断大趋势是牛是熊。
2. 用**差异指数**看看当下是否处于极端状态（超买或超卖）。
3. 在趋势方向上，用**蜡烛图**找具体的入场和离场信号（记住：蜡烛图不给目标价位，结合前支撑/阻力或斐波那契来确定离场位）。
4. 进场前，用「实体群组合判断」快速扫描横盘区的多空力量对比。
5. 计算好止损位和风险收益比——不划算就不做。
6. 形势变了就认——做一只市场变色龙。
