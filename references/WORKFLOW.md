# 统一交易引擎 — 完整工作流

## 命令速查

```bash
# 1. 实时分析
python /tmp/trading_engine.py --analyze AVAXUSDT

# 2. 批量回测
python /tmp/trading_engine.py --backtest

# 3. JSON 输出（接其他系统）
python /tmp/trading_engine.py --backtest --json > report.json

# 4. 扫描信号（模拟盘机器人）
python /tmp/trading_bot.py --simulate

# 5. 模拟盘下单
python /tmp/trading_bot.py --trade --coin AVAXUSDT \
  --api-key KEY --api-secret SECRET
```
