> 基于安德烈亚斯·克列诺（Andreas F. Clenow）《趋势交易——跟随趋势的分散化期货交易》（Following the Trend）

## 第一部分：底层思维——趋势跟踪的世界观

### 1.1 什么是趋势跟踪

趋势跟踪的核心哲学极其简单：**价格会沿着已经形成的方向继续运行。** 不需要预测市场，不需要分析基本面，只需要耐心等待向某个方向动量的积累，然后顺势而为。

你只需要做三件事：
1. 等——等到价格出现明确的上涨或下跌趋势
2. 跟——押注价格会朝同一方向继续运行
3. 退——当趋势反转、价格朝不利方向运行到一定程度时退出

**趋势跟踪者不预测未来，只对已经发生的变化做出反应。** 他们不是预言家，而是统计概率的利用者：在三笔交易中可能有两笔失败，但最后那一笔赚的钱要超过前两笔的亏损及所有费用。

### 1.2 和传统投资方法的本质区别

传统投资的信条是"低买高卖"、"买入并长期持有"，认为股市长期一定上涨。趋势跟踪则完全不同：

| 维度 | 传统投资 | 趋势跟踪 |
|------|---------|---------|
| 核心理念 | 预测价值、等待价值回归 | 跟随价格、不预测方向 |
| 买卖方式 | 低买高卖 | 高买更高卖，低卖更低买 |
| 对亏损的态度 | 亏损是暂时的，坚持等待 | 亏损必须迅速截断 |
| 盈利方式 | 靠选对资产 | 靠让利润奔跑 |
| 胜率 | 期望高胜率 | 胜率可能只有30%-42% |
| 与股市关系 | 高度相关 | 无关甚至负相关 |

趋势跟踪策略常常在"非常不合理"的价位开仓——买入已经涨了很多的品种，卖空已经跌了很多的品种。这让新手感到极度不适，但这恰恰是利润的真正来源。

### 1.3 为什么趋势跟踪能赚钱

**行为金融学基础**：市场由人组成，而人容易过度反应——恐慌时卖得太低，狂热时买得太高。当趋势出现时，群体的情绪会自我强化，推动价格沿着趋势方向运行得比理性预期的更远。趋势跟踪者搭的就是这趟顺风车。

**大数定律**：只要在足够多的品种上、足够长的时间里不断重复交易，统计优势就会显现。单次交易的结果不重要，重要的是长期期望值为正。

**厚尾效应**：趋势跟踪策略的收益分布呈现出严重的右偏——绝大多数交易不亏不赚或小亏，但极少数交易（5%-10%）能产生巨大的盈利，远远覆盖所有亏损。这种厚尾不是风险，恰恰是利润的来源。

### 1.4 趋势跟踪的局限性（批评与回应）

**批评一："过去的数据不代表未来。"**
回应：趋势跟踪不追求数学上的确定性，而是挖掘统计概率中对我们有利的部分。没有永远有效的策略，但趋势是人类群体行为的结果，只要人类行为模式不变，趋势就会持续出现。

**批评二："高杠杆意味着高风险。"**
回应：杠杆和风险没有必然关系。100万美元的黄金期货和100万美元的欧洲美元期货，名义价值相同，但实际风险天差地别。正确的持仓限额方法关注的是实际波动率，而非名义价值。

**批评三："卖空有无限风险。"**
回应：期货卖空的最大亏损限于保证金账户的全部资金。足够的分散化加上合理的止损，使卖空的实际风险可控。

**批评四："最大回撤太大，持续时间太长。"**
回应：这是不可回避的现实。但同样程度的回撤，在股票"买入并持有"策略中也常见（2008年MSCI全球指数从高点跌了55%），而趋势跟踪策略从未亏掉过去15年的全部收益。


**批评五："存在存活者偏误——指数只留下表现好的基金。"**
回应：这种偏误在所有指数中普遍存在。标普500每年也会剔除表现不好的成分股。这种现象不会只针对期货策略，且本书的比较不单依赖指数分析，还比较了行业里一大批巨头的实际业绩。

## 第二部分：核心原则

### 2.1 趋势跟踪成功的五大关键因素

原书明确指出，以下五个因素远比具体的买卖规则更重要：

**（1）投资组合的分散化程度**
这是最关键的因素。必须涵盖多个品种、多个板块。只交易单一品种或单一板块是"自杀性投资"。有些品种甚至会连续多年下跌，但只要其他品种的盈利能超过亏损，就值得保留——分散化本身就是价值。

**（2）持仓限额**
必须根据每个品种的波动性正确计算持仓数量，让每个品种对投资组合的日均波动贡献大致相等。不能简单地等名义价值配置，否则波动大的品种会主导整个组合的涨跌。

**（3）投资的期限**
选择追踪多长周期的趋势——一两周，还是5-8周？不同时间跨度的选择会导致结果差异巨大。本书选择的是中期趋势（平均持仓6-7周）。

**（4）风险水平**
在风险和收益之间找到平衡点。风险因子调高，收益和回撤都会放大；调低则相反。关键是让自己和投资人都能接受。

**（5）单一策略还是多策略**
将同一种策略应用到不同的时间尺度上，或同时使用多个相似策略，可以平滑短期业绩波动。长期收益接近，但过程更平稳。

### 2.2 趋势跟踪的"秘密"——其实没有秘密

行业内绝大多数CTA基金使用的核心策略大同小异。差异只在于：
- 投资品种池的选择
- 时间尺度
- 风险因子
- 板块集中度


- 各大基金之间月度对数收益率的相关系数可高达0.8以上（连接基金与母基金级别的相关性），这从数据上验证了"所有人做的都是同样的事情"
- 通过调整时间尺度、风险因子和投资品种池，几乎可以复制绝大多数CTA基金的表现
**最大的误区是把精力浪费在无谓的细节上**，比如：
- 用指数移动平均线还是简单移动平均线？
- 用RSI还是随机指标？
- 如何优化指标参数？
- 要不要加滤波？

这些问题的答案对最终结果的影响微乎其微。越关注这些细节，越容易错过真正重要的是什么。

**完全不需要复杂的数学、一大堆技术指标或自创的"独门秘方"来构建有效的趋势跟踪策略。**

## 第三部分：操作体系

### 3.1 期货市场基础知识

#### 什么是期货？
期货是在未来特定时间以约定价格买卖标的资产的标准化合约。对于交易者，可以简单理解为：
- 看涨就买入（做多）
- 看跌就卖出（做空）
- 不用实际交割，在到期前平仓即可

#### 期货交易的关键特点

**保证金交易**：只需支付合约名义价值的一小部分（通常5%-15%）作为初始保证金即可开仓，这提供了杠杆但也带来了风险。

**逐日盯市**：每天交易结束后按结算价对持仓盈亏进行现金结算。盈利当天入账，亏损当天扣款。

**固定到期日**：每个合约都有交割日，长期持有时需要"展期"——在旧合约到期前平仓，同时开仓到期日更远的新合约。

**标准化合约属性**（需要掌握的）：
- 合约乘数/点价：每变动一个最小价格单位的价值
- 初始保证金：开仓所需最低资金
- 维持保证金：持仓期间账户必须维持的最低资金

#### 期货的板块划分（五大板块）

本书将全部期货品种分为五个板块，每个板块中选取的品种应当涵盖不同的子类别：

**（1）股票板块**：各类股票指数期货（标普500、纳斯达克100、欧洲斯托克50、日经225等）。特点是板块内品种相关性极高，尤其市场恐慌时。长期趋势跟踪的盈利能力一般，但极端行情下贡献巨大（如2008年卖空）。

**（2）利率板块**：各国国债期货（美国2年/5年/10年/30年期、德国Bund/Bobl/Schatz/Buxl、英国金边债券等），以及短期利率期货（欧洲美元等）。历史上是最大利润来源（受益于几十年利率下行），但在低利率环境下前景存疑。

**（3）外汇板块**：主要货币对期货（欧元、日元、英镑、瑞士法郎、澳元、加元等，均以美元计价）。多空方向取决于把哪种货币视为本币。趋势行情通常与美元强弱高度相关。

**（4）农产品板块**：谷物（玉米、小麦、大豆等）、软商品（咖啡、可可、糖、棉花、橙汁等）、肉类（活牛、瘦肉猪等）。历史上收益最为平滑稳健，从未出现过整年大幅亏损。

**（5）非农商品板块**：金属（黄金、白银、铜、铂金等）、能源（原油、天然气、取暖油等）。多空都能贡献利润，但波动较大。

### 3.2 趋势跟踪的核心交易策略

本书从两种最基本的策略出发，逐步改进，最终形成一个实用、可靠的完整策略。

#### 策略一：双均线交叉策略（最基本）

- 当短期均线（如50天）上穿长期均线（如100天）时，做多
- 当短期均线下穿长期均线时，做空
- 始终持有仓位，方向由均线交叉决定

#### 策略二：通道突破策略（最基本）

- 用过去N天（如50天）的最高价和最低价构成通道
- 价格突破通道上轨时做多
- 价格突破通道下轨时做空
- 在反向突破时才平仓并反手

#### 最终成型的核心策略（经过改进和组合）

完整的策略包含以下要素：

**开仓条件（趋势过滤器+突破信号）**：
- 只有当50日均线高于100日均线时，才允许开多仓
- 只有当50日均线低于100日均线时，才允许开空仓
- 在允许的方向上，当某一天的收盘价是过去50天内最高的收盘价时，在下一个交易日买入
- 当某一天的收盘价是过去50天内最低的收盘价时，在下一个交易日卖出/卖空

**平仓条件（移动止损）**：
- 多头仓位：止损价 = 开仓以来最高收盘价 - 3×ATR
- 空头仓位：止损价 = 开仓以来最低收盘价 + 3×ATR

**持仓限额（ATR仓位公式）**：
```
合约数量 = 账户资产总值 × 风险因子 / (ATR值 × 点价)
```
- 风险因子建议初始设为0.2%（20个基点）
- 取整时采用"宁错过、不做错"的保守原则
- 建仓后合约数量不变（不随ATR变化而调整）

**品种池要求**：
- 必须涵盖全部5个板块
- 每个板块不少于10个品种
- 总品种池建议50个以上
- 专业CTA通常用到约100个不同合约

#### ATR（真实波动幅度均值）详解

**单日真实波动幅度（True Range）**：
```
TR = max(H_t - L_t, |H_t - C_{t-1}|, |L_t - C_{t-1}|)
```
即当日最高最低价之差、最高价与前日收盘价之差的绝对值、最低价与前日收盘价之差的绝对值，三者取最大。

ATR是多个交易日TR的指数移动平均，用于衡量品种的"正常"日均波动范围。

ATR的两个用途：
1. 确定仓位大小（横向比较不同品种的波动性）
2. 设定止损位置（动态跟踪止损）


#### 补充：期货数据的汇率换算
由于不同品种的期货合约可能以不同货币报价（如伦敦金属交易所以美元报价但账户可能是英镑），持仓限额公式中分母上的ATR×点价结果必须换算成与账户本币相同的币种，才能正确计算合约数量。此外，结算盈亏时同样需要处理多币种问题——每个子账户的币种应与其交易品种的结算货币匹配，以避免不必要的汇兑成本。
### 3.3 交易执行的实务细节

#### 交易频率和持仓时间

- 平均持仓时间：6周零4天
- 盈利头寸平均持仓：10周零3天
- 亏损头寸平均持仓：仅2周
- 平均交易频率：约每周3笔（不计展期）
- 展期不算新交易，是同一仓位的延续

#### 盈利交易的分布特征

- 盈利交易占比：约42%
- 亏损交易占比：约58%
- 有些盈利能力极强的趋势跟踪策略胜率甚至不足30%，但它们能够盈利的关键在于赚钱交易的盈利远大于亏钱交易的亏损
- 盈利持续期可能很长（连续盈利14笔），亏损持续期同样可能很长（连续亏损24笔），两者都是正常的
- 亏损交易占比58%是盈利的必要结构——正是这些频繁的小亏损，换来了捕捉大趋势的权利
- 70%的交易盈亏在±0.5%以内
- 没有一笔交易亏损超过2%
- 但有15笔交易的盈利超过5%
- 收益分布正向偏度高达+2.1

**关键认知：胜率不到一半是完全正常的。赚钱的交易（少数大赢）带来的盈利远远大于亏钱的交易（多数小亏）。**

#### 多空交易的差异

- 纯多头交易表现相对平滑向上，回撤浅、收益稳定
- 纯空头长期收益能力差（年化仅2.5%，恢复期长达13年），但能在市场极端恐慌时贡献巨大收益
- **空头交易的价值不在于长期收益，而在于作为分散化工具和危机稳定器**
- 包含空头的策略与股市相关性为-0.16，纯多头策略则升为+0.18
- 去掉空头会降低产品对持有大量股票的客户的吸引力


**参考数据**：MSCI全球指数1990-2011年平均年化收益率仅4.7%（不含费用），最大历史回撤57.5%。在最严重的顶部买入后，需要10年才能收回本金。
#### 各板块的收益贡献特征

- **利率板块**：历史上最大利润来源，受益于长期利率下行。但在接近零利率的环境下前景不明。
- **非农商品**：多空都能赚钱，稳定性好，偶尔有爆发。
- **农产品**：收益最平滑稳定，从未有整年大亏。
- **外汇板块**：盈亏年份约各一半，盈利年份赚的多于亏损年份亏的。
- **股票板块**：盈利最少，做空甚至长期亏钱。极端行情下（如2008年）爆发力极强。

## 第四部分：风险管理与资金管理

### 4.1 持仓限额的核心理念

**目标**：让每个品种对投资组合的日均盈亏波动的贡献大致相等。

**为什么不能等名义价值配置？**
- 欧洲斯托克50指数一天波动1%-2%很常见，极端时可达4%-5%
- 欧洲美元一天波动0.5%闻所未闻
- 如果每个品种买相同名义价值，波动大的品种会完全主导组合走势

**ATR仓位公式**的核心思想：
- 波动性低的品种 → 合约数量多（提升其影响力）
- 波动性高的品种 → 合约数量少（降低其影响力）

### 4.2 风险因子——调节策略风险水平的唯一旋钮

风险因子是策略中最主观的参数。它的含义是：理论上每个品种在一个正常交易日内的价格变动对组合总价值的影响。

- 风险因子 = 0.2%：每品种日均影响2000美元（对100万美元组合）
- 风险因子 = 0.1%：低风险，低收益，低回撤
- 风险因子 = 0.25%：高风险，高收益，高回撤

**关键原则**：
- 通过模拟确定自己真正能接受的风险因子
- 宁可保守，不要激进
- 新基金成立初期尤其要保守——生存是第一要务
- 一旦风险因子确定，严格执行，不在压力下临时调整


#### 补充：可用保证金比率替代ATR做仓位控制
很多大型期货对冲基金只用"单笔头寸占用保证金/资产总值"的比率来控制持仓限额和衡量风险，同样取得了优秀的成绩。这种方法的理论基础是：交易所在设定保证金时已考虑了品种的波动性。缺点是：①交易所可能无预警地突然调整保证金标准；②用历史数据回测时难以获取完整的历史保证金记录。
### 4.3 止损机制

本书采用**动态跟踪止损**（Trailing Stop），而非固定点位止损：

- 多头：止损价随最高收盘价上移（最高收盘价 - 3×ATR）
- 空头：止损价随最低收盘价下移（最低收盘价 + 3×ATR）

这种止损方式的特点：
- 保护已获得的利润（止损价随价格有利方向移动）
- 给趋势留出"呼吸空间"（3个ATR的宽度可以容纳正常的反向波动）
- 约60个基点的止损距离，解释了为何单笔亏损最容易落在0.5%-0.75%区间

**滑点和跳空**：实际止损价可能劣于预设止损价，特别是出现跳空缺口时。保守估计每手双边手续费+滑点损失约20美元。

**止损距离与风险因子联动**：持仓限额公式与止损公式中都包含ATR——合约数 = 资产总值 × 风险因子 / (ATR × 点价)，止损距离 = 3 × ATR。因此每笔交易理论上的最大亏损 = 风险因子 × 3 ≈ 0.2% × 3 ≈ 0.6%的总资产。这解释了为何单笔亏损最频发区间在0.5%-0.75%（加上滑点偏差）。

**重要提示**：止损信号在收盘价确认后才触发，而非盘中即时。实际平仓发生在信号出现后的下一个交易日开盘。如果出现跳空缺口，实际止损价可能远劣于预设止损价。

### 4.4 现金管理——被低估的利润来源

期货交易不需要像股票那样支付全款，账户中有大量闲置现金。正确的现金管理是：

**将闲置现金配置在政府债券上（而非存在银行）**，因为：
1. 获取利息收入（20世纪八九十年代可贡献年化+5%-6%的收益）
2. 降低交易对手风险——经纪商破产时，证券类资产能被完整返还，但现金可能血本无归

**每个子账户保留的现金应**：
- 足以覆盖每日结算的亏损
- 足以支付可能的客户赎回
- 除此之外尽量少——余钱都买国债

**低利率时代的冲击**：2012年之后利息收入几乎为零，趋势跟踪策略失去了这层"安全垫"，整体收益率有所下移。

### 4.5 杠杆的正确理解

**错误认知**：高杠杆 = 高风险
**正确认知**：

合约名义价值与实际风险完全是两回事。100万美元的黄金期货和100万美元的欧洲美元期货，名义杠杆相同，但实际价格波动相差数十倍。只要持仓限额根据波动性正确计算，使用保证金交易带来的高名义杠杆并不等于高风险。实际上，合理的杠杆是实现充分分散化的必要条件——不用杠杆就无法同时参与几十个品种。


#### 补充：复利效应与杠杆的误区
人们在放大策略杠杆时有一个常见误区——忽视了复利效应的影响。原书通过一个虚拟例子证明：增加一倍杠杆并不等于增加一倍的最终收益。由于复利效应的存在，高杠杆下的累计收益会出现非线性的变化。
### 4.6 相关性风险

**股票板块内部的"虚假分散化"**：不同国家、不同行业的股指期货在市场恐慌时相关性能快速趋近于1。看似分散的持仓可能在极端情况下突然变得高度一致，导致短时间内出现严重亏损。

**同主题持仓的隐性集中**：如同时做空股票+做多利率+做空美元，表面上分散在不同板块，实际上都押注同一个宏观主题（风险偏好下降），一旦主题反转，所有头寸可能同时亏损。

## 第五部分：心理纪律

### 5.1 趋势跟踪的心理挑战

趋势跟踪是**反人性的**。以下是交易者必须面对的真实心理困境：

**挑战一：大多数交易都在亏钱**
58%的交易以亏损告终。连续亏损14笔是正常现象。连续亏损24笔也是正常现象。趋势跟踪策略的盈利靠的是少数的"大赢"，而非多数的"小赢"。

**挑战二：在高位买入、低位卖空**
买入时价格往往已经涨了很多，卖空时已经跌了很多。这会让人觉得"买贵了"或"卖便宜了"，极度不适。

**挑战三：盈利反复回吐**
当账面上有+36%的浮盈，然后眼睁睁看着它回落到+5%——这种事情在趋势跟踪中经常发生。策略的特点就是让利润奔跑，然后等趋势反转时回吐一部分。

**挑战四：漫长的回撤期**
可能连续两年没有像样的收益，还可能遭遇-20%的最大回撤。期间没有业绩报酬可收，生活可能发生巨大改变。

**挑战五：对巨额亏损的心理承受**
模拟中看到的15%回撤只是一个数字，但当你亲眼看到账户每秒亏损成千上万美元时，那种压力完全不同。在这种状态下做决策通常是灾难性的。

### 5.2 应对心理挑战的策略

**（1）把交易当成游戏，而非钱财**
不要将基金资产看作真正的金钱，只当作一个赚取积分的游戏。如果总是把亏损换算成现实中能买到的东西，就会丧失最后的洞察力。

**（2）绝对不要在压力下做决定**
一定要在事前制定好规则：什么情况下可以变更策略、变更的前提条件和具体方式。危机发生时，如果没有提前准备好的计划，继续遵循原策略就是最明智的选择。

**（3）经历是最好的老师**
大多数人只有亲身经历困难时期、甚至多次遭遇危机之后，才能慢慢习惯。经历过1994年那种-17%的惨败后还能坚持下来，并且第二年重新盈利，对策略和自身的信心会升华到全新的高度。

**（4）不要过度盯盘**
对于基于日线收盘价产生信号的策略，终日盯着日内波动毫无意义，只会加深焦虑。克制盯盘的冲动是必须培养的能力。

**（5）与客户保持良好沟通**
无论是好业绩还是坏业绩，都要让投资人了解策略的大致情况、当前风险水平、持仓近况和板块分布。知悉情况的客户更有可能陪你度过困难时期。

**（6）基金经理跟投的利与弊**
将个人身家投入基金会加重心理负担，让保持"游戏心态"变得更难。保持与所管理资产的一定距离，有助于以更冷静的心态对待策略。

#### 有效边界的关键结论
从原书构建的两类资产（期货策略+MSCI全球指数）的有效边界分析中得出：在期货策略上的资金配置**低于30%-40%是不合理的**——低于这个比例意味着投资组合获得更低的收益，却没能降低风险。增加股票持有比例虽然会降低总回报，但可以减少最大回撤的深度。

### 5.3 历史教训——你必须知道的事

**1994年（最惨的一年）**：
- 基金全年亏损17%
- 1月第一周就亏了5%，2月底亏13%，11月触及-20%
- 多头交易是罪魁祸首（-12%），空头也亏了5%
- 主要亏损来自非农商品和股票板块
- 全年反复出现"假突破"，如图30中的黄金——在最低点卖空，价格迅速拉升后坐实亏损
- 同期MSCI全球指数小幅上涨，加剧了投资人的不满

**2004年（大喜大悲的一年）**：
- 3月浮盈+16%，到9月变成-9.5%
- 一路亏掉全部资产的25%
- 最终年底只剩+3%
- 暴跌由多个板块共同作用：利率多仓获利回吐+股票多仓持续亏损+新建利率空仓的损失

**2008年（梦幻般的一年）**：
- 全年盈利108%
- 10月单月盈利从+30%暴增至+80%
- 利率多头贡献了基金年初净值40%的利润
- 但过程同样惊险：经纪商倒闭、客户资金安全岌岌可危
- 很多投资经理在极度波动中人为干预了策略，错过了部分利润

**关键启示**：
- 你永远不知道哪年会赚大钱，哪年会赔大钱
- 唯有坚持策略、不做人为干预，才能确保在"大年"到来时在场
- 最差的年份（1994年：-17%）和最好的年份（2008年：+108%）都可能出现
- 22年中只有1994年出现两位数亏损，是策略韧性的最好证明

**更多值得铭记的年份**：

- **1990年**（策略启动年）：伊拉克入侵科威特，油价暴涨。年初组合严重偏向空头，在原油等品种上出现大幅亏损。但随后趋势反转，利息收入贡献高达8.7%。
- **1991年**（苏联政变年）：苏联"八一九事件"引发全球市场动荡，但未造成实质伤害。全年收益25.6%，年末峰回路转主要归功于利率多头——苏联解体后西方利率开始稳步下降。
- **1993年**（农产品爆发年）：全年盈利超40%。核心驱动力是农产品板块（特别是瘦肉猪期货，从2月开始如火箭般上涨）。8月在非农商品上遭遇"双杀"——同时做多金属+卖空能源，两者同时反转导致一天内损失惨重。
- **1998年**（LTCM崩盘年）：俄罗斯经济危机引爆长期资本管理公司崩溃。全年盈利超40%，趋势跟踪者在秋冬季突然加速获利，几天内冲上+42%。
- **2001年**（9/11年）：全年盈利高达36.2%。年初就持有大量针对股市熊市的头寸（卖空8个股指、做多12个利率期货）。"9·11"后几乎所有头寸同时贡献巨额利润。
- **2003年**（美元大跌年）：全年盈利28.7%。最大利润来自外汇多头（本质是做空美元）。
- **2006年**（极度波动年）：年初持有全部10个股票多仓。5月底净利+36%，随后全部10个股票多头被止损。从+35%暴跌至+5%，最终年底+20%。结果虽好但过程极为痛苦。
- **2007年**（中国股灾波及年）：2月27日中国股市单日跌近10%，余震波及全球。一天内全部股票多仓被止损，几天内总损失达15%。最终依靠农产品和非农商品在年底获+8%。

## 第六部分：策略的改进与深化

### 6.1 同时捕捉不同时间尺度的趋势

原策略使用50天突破判断中期趋势。可以将同一策略同时运行在多个时间尺度上（如25天、50天、100天），各按比例分配资金。

- 25天：收益更高，波动更大
- 100天：收益略低，但更平滑
- 组合使用可以显著平滑短期业绩波动

### 6.2 合成期货的交易

某些品种只有远期合约流动性好，或不同合约月之间价差异常大。可以通过交易"合成期货"——即用多个不同月份的合约组合模拟出一个虚拟的连续价格序列——来改善策略表现。

### 6.3 添加逆势交易的元素

趋势跟踪策略在价格突破时建仓。正因为大量趋势跟踪者同时涌入，突破后的价格可能在短期内偏离合理范围。逆势交易者专门利用这种短期过度反应获利——这就是从趋势跟踪者口中夺食。

为规避这种风险，可以：
- 不在突破后立即全额建仓，而是分步建仓
- 使用多时点手工分批建仓
- 关注逆势交易者的存在，在策略设计中留有余地

### 6.4 日内止损的考量

原策略使用收盘价作为止损触发条件。日内止损意味着在盘中价格触及止损位时就平仓，而非等到收盘。主要考量：
- 优势：能更快地截断亏损
- 劣势：容易被盘中"异常波动"（如闪电崩盘）洗出，错失后续趋势

需要根据自己的品种池和市场环境权衡。对于波动剧烈、经常出现日内极端反转的市场，日内止损可能弊大于利。

### 6.5 相关性矩阵与持仓限额

当投资组合中多个品种高度相关时（如同时做多标普500、纳斯达克100、欧洲斯托克50），表面上每个品种仓位正常，实际上整个板块的风险被放大了数值。更精细的做法：

- 构建品种之间的相关性矩阵
- 当多个品种相关性超过阈值时，降低该"主题"的总仓位
- 考虑板块级别的总风险上限

### 6.6 展期效应

期货合约有固定的到期日，展期是必须的。但展期时旧合约和新合约之间存在价差，这个展期成本（或收益）会侵蚀（或增厚）策略收益。

- 正向市场（期货升水）：远月合约价格高于近月，多头展期吃亏，空头展期获利
- 反向市场（期货贴水）：远月合约价格低于近月，多头展期获利，空头展期吃亏

在模拟时需要考虑展期效应，使用连续数据时需要确认数据的构建方式是否正确。

### 6.7 模拟优化的陷阱

**过度优化**是最大的陷阱。不要使用任何软件的"参数优化"功能来寻找"最佳参数"——这只不过是在拟合历史曲线，对未来毫无预测力。

正确的做法：
- 设定合理的参数范围（如突破天数在25-100天之间）
- 进行**参量稳定性检验**：验证策略在参数调整后表现是否大体一致
- 如果某个参数微调就导致结果剧烈变化，说明策略不稳定，不可靠
- 本书策略通过了稳定性检验（25天/50天/100天结果相近）

**保守地估计成本**：在模拟中将滑点损失、手续费等成本设得比实际预期更高，宁可低估收益、不要高估。

### 6.8 资金规模的限制

趋势跟踪策略具有很好的弹性——同样策略在相同操作手法下可承受资产膨胀到相当大。但存在两个实际限制：

1. **流动性上限**：当管理资产达到5-10亿美元时，流动性稍差的品种就开始难以参与了。小品种的流动性无法容纳大资金量。
2. **收益递减**：随着资金规模增长，不可能继续承受同样的风险水平——这也是大型老牌基金收益率逐渐下降的原因之一。对于管理规模超过50亿美元的基金，能否保持25%+的年化收益率已经不是最值得关注的事。
## 第七部分：实盘运营与商业考量

### 7.1 以管理期货为生的商业模式

管理期货（CTA）的商业模式本身非常有吸引力：

**收入来源**：
- 管理费：每年按管理资产规模的1%-2%收取
- 业绩报酬：正收益部分的10%-20%（通常15%）

**核心优势——规模弹性**：
- 管理2000万美元与管理1000万美元的工作量几乎相同
- 同样的策略、同样的操作，只是每笔合约数量翻倍
- 一般在管理资产达到5-10亿美元时才会遇到流动性的新挑战

**典型收入示例**：
自有10万美元+客户100万美元，20%年化收益：
- 自有资金收益：2万美元
- 管理费（1.5%）：1.5万美元
- 业绩报酬（15%）：3万美元
- 总收入：6.5万美元（从2万变成6.5万）

### 7.2 两种运营结构

**定向账户（Managed Account）**：
- 客户以自己名义开立账户，委托交易员操作
- 客户保留随时监控和干预的权力
- 法律结构简单、安全性高
- 客户黏性更强
- 但行政管理工作量更大，参与资金起点更高

**对冲基金**：
- 所有客户资金集中在一个大账户中交易
- 信息披露和档案管理简化
- 需要更复杂的法律架构
- 客户黏性相对较低

### 7.3 处理申购和赎回

- 理论正确做法：按申购/赎回金额等比例调整所有头寸
- 小规模组合的实操问题：如果只有3-4手合约，15%的资金变化无法等比例调整
- 解决方法：优先用新增资金弥补之前因取整而被低配的资产；优先变现高配资产应对赎回

**期货策略的优势**：账户中总有足够的现金用于兑付正常赎回，一般不需要被迫变现持仓。

### 7.4 波动性与营销

**波动性（风险）是为获取收益必须支付的成本。**

- 客户不愿接受与高波动不匹配的高收益
- 夏普比率不应低于0.5，索提诺比率是更好的补充指标
- 行业内默认：连续两年业绩不佳，第三年还不行就基本散伙了
- 定向账户客户比对冲基金客户更具黏性（因为有更多直接交流）

**业绩报酬的"高水位规则"**：
- 必须先补足之前的亏损，才能对剩余收益收取业绩报酬
- 这导致：经历大幅回撤后，即使当年收益不错，投资经理能收到的业绩报酬也很少
- 同时也导致：亏损时期业绩波动反而更大（因为没有被"未提业绩报酬"缓冲）

### 7.5 启动资金的现实困境

- 找到初始启动资金是最困难的一步
- 需要准备可信的产品方案：明确的投资策略+执行能力+独特优势
- 通常需要借助圈子里的朋友和人脉
- 有些人被迫接受"不平等条约"（向介绍方支付返佣）
- 2008年之后上马的基金：很多在2009-2010年被"双杀"——成立即遇困难期，亏损20%-25%后清盘

### 7.7 交易执行中的实务难题

趋势跟踪策略在实际执行中会面临一些模拟中难以体现的困难：

- **流动性问题**：某些期货品种在特定时间可能突然失去流动性，导致无法以合理价格开仓或平仓。
- **跳空风险**：重大事件可能导致价格直接跳过止损位开盘，使实际止损价远劣于预设。尤其在隔夜或周末持有仓位时风险最大。
- **合约选择**：应始终选择流动性最好的合约月份进行交易。通常近月合约流动性最好，但临近交割日时需及时展期到远月。
- **展期操作**：在旧合约流动性开始枯竭前就需要展期，而不是等到最后一刻。展期时新旧合约之间的价差是交易成本的一部分。
- **闪电崩盘风险（2010年）**：2010年发生的"闪电崩盘"对使用杠杆的趋势跟踪者是致命的。此类事件以前几乎从未发生过，提醒我们市场结构正在变化，日内极端波动可能骤然出现。
## 第八部分：数据、工具与技术

### 8.1 数据的重要性

**期货数据的特殊性**：
- 不能直接使用未处理的原始合约数据
- 必须构建连续数据（处理展期）
- 数据处理方式会直接影响模拟结果
- "用错误的数据构建策略就是在浪费生命，基于错误数据交易是在挥霍财富"

**数据处理方式**：
- 展期处理（后顾调整 vs 前瞻调整）
- 成交量的考虑
- 数据提供商的选取（CSI Data、彭博等）

**自建本地数据库的优势**：
- 不受制于单一数据提供商
- 可随时更换提供商或综合多来源数据
- 避免数据提供商周末维护时无法获取数据
- MySQL或类似数据库搭建成本低

### 8.2 编程能力——不可或缺

**最低要求**：熟练一门通用编程语言（推荐C#）

**为什么不能用现成的脚本语言？**
- 脚本语言局限性大，受限于条条框框
- 复杂策略很快就会发现能力不够
- 目标是专业人士，不是散户

**好的开发环境标准**：
- 基于标准化语言（非自创脚本语言）
- 能处理海量数据和内存占用高的组合策略
- 允许DLL动态链接库等外挂
- 64位系统

**推荐工具**：WealthLab.Net（易学）、RightEdge（更灵活）、MatLab

### 8.3 模拟与回测

**必须计入的成本**：
- 手续费+滑点：每手双边约20美元（保守估计）
- 管理费：每年1.5%
- 其他费用（托管、行政）：0.5%
- 业绩报酬：正收益的15%
- 利息收入：按历史利率计算

**模拟中必须回答的问题**：
- 最大回撤的幅度和恢复时长
- 在不同风险因子下的表现
- 各板块、多空分别的贡献
- 策略参量的稳定性

## 第九部分：策略的监控与后续管理

### 9.1 实盘与模拟的对比跟踪

实盘一旦启动，最重要的工作就是将实际结果与模拟预期做持续比较：

- 记录每日实际净值变化和总费用
- 与模拟计算的日度结果对比
- 如果连续几个月实盘低于预期，可能存在未知缺陷
- 最好能将每个品种的实盘单日盈亏与模拟预估做比较

### 9.2 投资组合监控的悖论

对于使用日线收盘价产生信号的策略：**实时盯盘弊大于利**。

- 日内盈亏波动会增加焦虑
- 焦虑导致不当干预
- 但几乎所有人都忍不住——包括作者自己

### 9.3 行业趋势与未来挑战

**收益能力下降的两大原因**：
1. 利息收入减少——政府债券利率从5%-6%降至接近零
2. 市场行为改变——电子化交易和高频算法交易盛行，趋势周期变短，反向调整更剧烈，群体自相踩踏加剧

**应对之道**：
- 核心策略仍然有效，但要有清醒认识
- 持续研究、时刻警惕市场变化
- 永不停止对策略的改进和学习

### 9.4 最后的忠告——初始风险水平

**生存是第一要务。**

- 新基金成立初期风险水平要保守——即使错过一些盈利
- 历史上大量新基金在好时光后成立，紧接着遭遇困难期而清盘
- 运气不好时一两年内亏掉20%-25%不是不可能
- 只有老牌基金能熬过困难期
- 宁可在成立时保守一些，活下来才有机会


**初始风险管理的时间维度**：大多数新基金在趋势跟踪策略表现优异之后成立（如2008年之后），紧接着就遭遇困难期。如果2009年初投入一只典型趋势跟踪基金，随后将面临-20%以上的回撤。没有长期业绩记录的新基金几乎没有熬过困难期的可能。**起步时风险减半、生存优先**——即使这意味着错过一些盈利机会。在管理资产增长到有足够"安全垫"后（如积累了几年的正收益和稳定的管理费收入），再逐步提升风险因子。

### 9.5 历史重要事件启示

以下历史事件对趋势跟踪策略有重大影响，了解它们有助于增强对策略的信念：

| 年份 | 事件 | 策略表现 | 关键教训 |
|------|------|---------|----------|
| 1990 | 伊拉克入侵科威特 | 约+10% | 年初严重偏空导致亏损，但趋势反转带来盈利 |
| 1991 | 苏联解体 | +24% | 利率下行开启了债券牛市的黄金时代 |
| 1994 | 全球市场震荡 | -17% | 假突破频发、多板块同时亏损；最惨一年 |
| 1998 | 俄罗斯危机/LTCM崩盘 | +40% | 危机孕育大趋势；趋势者的"梦想之年" |
| 2001 | 9·11事件 | +36% | 空头多头在极端事件中同时获利 |
| 2003 | 美元持续下跌 | +29% | 外汇板块成为年度最大利润来源 |
| 2006 | 商品牛市中段 | +20% | 极度波动，+36%→+5%，考验意志力 |
| 2007 | 中国股灾/全球震荡 | +8% | 一天内亏15%，分散化配置救回全年 |
| 2008 | 全球金融危机 | +108% | 百年一遇；利率多头贡献了40%利润 |
| 2009 | 危机后反弹 | -12%（估计） | 好时光后立即遭遇困难——新基金在此年大量清盘 |
| 2010 | 闪电崩盘 | 震荡 | 高频交易对趋势者冲击加剧 |
| 2011 | 欧债危机 | 震荡 | 市场波动加剧但缺乏持续趋势 |

### 9.6 破产经纪商的警示

2008年贝尔斯登、雷曼兄弟、全球曼氏金融（MF Global）相继倒闭，大量客户现金血本无归。重要教训：

- 选择经纪商时做好尽职调查，优选只有经纪业务、无自营投资的券商
- 同时使用多个经纪商——一旦其中一个出问题，可快速转移现金和持仓
- 账户中尽可能少留现金，多余资金配置在短久期国债上
- 警惕"隔离专户"的虚假安全感——现实中往往并非如经纪商所说的那样安全
## 第十部分：自查清单

### 10.1 策略构建自查清单

- [ ] 品种池是否覆盖全部5个板块？（股票、利率、外汇、农产品、非农商品）
- [ ] 每个板块是否至少有10个不同品种？
- [ ] 总品种池是否达到50个以上？
- [ ] 数据是否经过正确的展期处理？
- [ ] 持仓限额公式是否正确使用ATR和风险因子？
- [ ] 开仓条件是否包含趋势过滤器（均线过滤）+ 突破信号？
- [ ] 止损是否使用动态跟踪止损（ATR倍数）？
- [ ] 风险因子是否通过模拟确定了适合自己承受能力的值？
- [ ] 策略是否通过了参量稳定性检验（调整关键参数后表现大体一致）？
- [ ] 模拟是否计入了所有成本？（手续费、滑点、管理费、业绩报酬）
- [ ] 是否考虑了展期效应和利息收入？
- [ ] 是否分别分析了多空交易的收益贡献？
- [ ] 是否分别分析了各板块的收益贡献？

### 10.2 实盘启动前自查清单

- [ ] 初始风险因子是否足够保守？
- [ ] 现金是否大部分配置在政府债券上？
- [ ] 各子账户的币种现金是否足以应付日常结算和赎回？
- [ ] 是否准备了产品推介材料（策略理念、风险收益特征、独特点）？
- [ ] 是否与潜在投资人充分沟通了策略的可能回撤幅度和恢复周期？
- [ ] 是否确立了申购/赎回的处理流程？

### 10.3 实盘运行中自查清单

- [ ] 是否在持续对比实盘表现与模拟预期？
- [ ] 是否避免了在压力环境下临时修改策略？
- [ ] 是否与投资人保持定期透明沟通？
- [ ] 是否克制了过度盯盘的冲动？
- [ ] 是否定期审视各板块、多空的收益贡献？
- [ ] 是否关注了经纪商的对手风险？

### 10.4 心理纪律自查清单

- [ ] 是否接受"大多数交易都会亏损"这一事实？
- [ ] 是否做好了在高位买入、低位卖空的心理准备？
- [ ] 是否准备好了面对大幅浮盈回吐时的焦灼？
- [ ] 是否准备好了连续两年零提成的可能性？
- [ ] 是否能把交易当作"游戏"而非真实的钱财？
- [ ] 是否在遭遇-15%以上的回撤时还能坚持原策略？

## 附录A：策略核心公式汇总

**真实波动幅度（TR）**：
```
TR = max(H_t - L_t, |H_t - C_{t-1}|, |L_t - C_{t-1}|)
```

**持仓合约数量**：
```
N = floor(账户总值 × 风险因子 / (ATR × 点价))
```

**多头止损价**：
```
止损 = 开仓以来最高收盘价 - 3 × ATR
```

**空头止损价**：
```
止损 = 开仓以来最低收盘价 + 3 × ATR
```

**开多条件**：50日均线 > 100日均线 且 收盘价 = 过去50日最高
**开空条件**：50日均线 < 100日均线 且 收盘价 = 过去50日最低
