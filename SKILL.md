# 交易大师体系 — 融会贯通七大经典

基于7本经典交易著作的完整知识融合，形成从市场认知→技术分析→交易系统→风险管理→心理修炼的完整闭环。

## 知识结构总览

本体系将7本书的知识重新组织为**五层金字塔**：

| 层级 | 名称 | 核心问题 | 支撑书系 |
|------|------|---------|---------|
| L1 | 市场认知 | 市场如何运作？我如何看市场？ | 道氏理论(Livermore)、五大基本事实(Douglas) |
| L2 | 技术分析 | 价格在说什么？趋势往哪走？ | K线(Nison)、形态/指标/波浪(Murphy) |
| L3 | 交易系统 | 如何系统化地捕捉机会？ | 趋势跟踪(Clenow)、海龟法则(Faith) |
| L4 | 风险管理 | 如何活下来并让利润奔跑？ | 资金管理(Murphy)、ATR仓位(Faith+Clenow) |
| L5 | 心理修炼 | 如何战胜自己？ | 概率思维(Douglas)、自我修炼(Livermore) |

## L1：市场认知 —— 建立正确的世界观

### 道氏理论三大基石（Murphy Ch2 + Livermore）

1. **市场行为包容消化一切** — 价格说明一切，不用猜新闻
2. **价格以趋势方式演变** — 趋势是你的朋友，顺势而为
3. **历史会重演** — 人性不变，市场模式不变

### 三种趋势（道氏理论核心）

- **主要趋势**（潮汐）：持续一年以上，大方向
- **次级趋势**（波浪）：持续三周到三个月，主要趋势中的回调
- **短暂趋势**（涟漪）：几天内的小波动，可忽略

**核心原则**：先判断主要趋势方向，然后只朝那个方向交易。

### 最小阻力路线（Livermore核心概念）

市场沿着最小阻力方向运行。判断方法：
- 上涨中回调浅、时间短、继续创新高 → 最小阻力向上
- 下跌中反弹浅、时间短、继续创新低 → 最小阻力向下

### 市场的五个基本事实（Douglas）

1. 任何事都可能发生
2. 不必知道下一步就能赚钱
3. 任何优势产生的赢亏分布是随机的
4. 优势只表示概率较高
5. 市场中的每一刻都是独特的

**关键推论**：你无法预测市场。唯一能做的是在概率有利时下注，在不利时退出。

## L2：技术分析 —— 读懂价格语言

### 趋势分析（Murphy Ch4）

**趋势定义**：一系列越来越高的高点和低点（上升），或越来越低的高点和低点（下降）。

**关键工具**：
- **趋势线**：沿回调低点向上画（上升），沿反弹高点向下画（下降）。45°最健康。触及次数越多越有效。被突破后互换角色。
- **管道线**：与趋势线平行，形成通道。触及管道线是顺势加仓/减仓位。
- **百分比回撤**：38.2%（强趋势）/ 50%（正常）/ 61.8%（弱趋势）。超62%警惕反转。

**支撑和阻挡**（最重要的概念）：
- 形成原因：锚定效应+近期偏好+处置效应（Faith）
- **支撑和阻挡会互换** — 跌破的支撑变阻挡，涨破的阻挡变支撑
- 判断强度：在那个位置成交越多、停留越久、离现在越近，越有效
- 整数关口有心理效应
- **价格不稳定点**（Faith）：支撑/阻力附近，要么守住反弹、要么跌破加速，都是好交易机会

### 反转形态（Murphy Ch5）

| 形态 | 特征 | 确认条件 | 测算目标 |
|------|------|---------|---------|
| **头肩顶** | 左肩→头→右肩，交易量递减 | 跌破颈线 | 头到颈线距离，从跌破点投射 |
| **头肩底** | 头肩顶倒影，底部更磨 | 涨破颈线+放量 | 同上 |
| **双重顶(M顶)** | 两高点相近，第二顶量小 | 跌破中间谷底 | 颈线到顶的距离 |
| **双重底(W底)** | 两低点相近 | 涨破中间峰顶+放量 | 同上 |
| **三重顶/底** | 三个相近高/低点 | 同双重 | 力度通常大于双重 |
| **圆底/圆顶** | 锅形/倒碗，交易量先缩后放 | 无明确突破点 | 相对模糊 |
| **V形反转** | 急跌急涨，无过渡 | 反转日+巨量 | 需配合趋势线和均线 |

**共同特征**：必须先有趋势、趋势线先被突破、形态越大后市越大、顶部快底部磨、突破必须放量。

### 持续形态（Murphy Ch6）

| 形态 | 特点 | 方向 | 测算 |
|------|------|------|------|
| **对称三角形** | 高点和低点向中间收敛 | 沿原趋势 | 底边高度投射 |
| **上升三角形** | 水平上边+上升下边 | 向上 | 同上 |
| **下降三角形** | 水平下边+下降上边 | 向下 | 同上 |
| **旗形/三角旗** | 小平行四边形，1-3周 | 沿原趋势 | 旗杆高度 |
| **楔形** | 两线同向倾斜 | 与倾斜相反 | — |
| **矩形** | 两水平线间震荡 | 沿原趋势(多数) | 矩形高度 |

**规则**：交易量在形态内萎缩，突破时必须放量。突破应在水平宽度的1/2到3/4处。不要在形态内部交易。

### 移动平均线（Murphy Ch9）

**单均线**：价格在线上=多头，线下=空头。线上回调+反转K线=买入机会。

**双均线交叉**：短期上穿长期=金叉（买入），下穿=死叉（卖出）。

**MACD**（12,26,9）：DIF上穿DEA买入，下穿卖出；零轴分强弱；柱状图背离是强信号。

**布林带**：触及上轨=超买，下轨=超卖；带收窄=即将大行情。

### 摆动指数（Murphy Ch10）

| 指标 | 超买 | 超卖 | 最强信号 |
|------|------|------|---------|
| RSI(14) | >70 | <30 | 背离+超买/超卖区 |
| 随机KDJ | >80 | <20 | 背离+K/D交叉 |

**黄金法则（极其重要）**：
- 上升趋势中：摆动指数的**超卖信号是买入时机**（不是卖出！）
- 下降趋势中：摆动指数的**超买信号是卖出时机**（不是买入！）

### 交易量和持仓兴趣（Murphy Ch7）

**价格 > 交易量 > 持仓兴趣**（重要性排序）

| 场景 | 应有交易量 | 含义 |
|------|-----------|------|
| 上涨 | 放大 | 健康 |
| 上涨 | 萎缩 | 背离，警惕 |
| 下跌趋势 | 放大 | 抛售强 |
| 回调 | 萎缩 | 健康 |
| 形态突破 | 必须放大 | 否则假突破 |

### 周期分析（Murphy Ch13）

市场存在不同长度的周期（5天、10天、20天、28天、9周、18周、9个月等）。周期叠加、谐波相关。从上一低点用周期长度投射下一时间窗口，在该窗口内密切观察价格确认信号。

### 蜡烛图技术（Nison全书）

#### 基础知识
- **长白实体** = 多头主控，自身构成支撑（实体中点以下影线底）。
- **长黑实体** = 空头主控，自身构成阻力（实体中点以上影线顶）。
- **纺锤线** = 多空拉锯，趋势动能衰竭。
- **十字星** = 市场犹豫，埋下反转伏笔。等下一根确认方向。
- **长上影线** = 高价被拒绝；**长下影线** = 低价被抢购。

#### 单线反转形态
| 形态 | 位置 | 含义 | 确认 |
|------|------|------|------|
| 锤子线 | 下跌后 | 看涨 | 下影≥实体3倍，上影极短 |
| 吊颈线 | 上涨后 | 看跌 | 形态同锤子，**等次日收盘确认** |
| 流星线 | 上涨后 | 看跌 | 上影≥实体3倍，实体在底部 |

#### 双线反转形态
| 形态 | 构成 | 位置 | 要点 |
|------|------|------|------|
| 乌云盖顶 | 长白+高开低走长黑，切入白实体中点以下 | 上涨后 | 切入越深越强 |
| 刺穿形态 | 长黑+低开高走长白，切入黑实体中点以上 | 下跌后 | 同上 |
| 多头吞噬 | 长白完全包住前一根小黑 | 下跌后 | 强于刺穿 |
| 空头吞噬 | 长黑完全包住前一根小白 | 上涨后 | 强于乌云盖顶 |
| 包孕形态 | 长实体内含小实体 | 趋势后 | 动能衰竭 |

#### 三星形态
- **黄昏星**：长白→小实体（跳空）→长黑（切入长白）。顶部反转，最可靠之一。
- **启明星**：长黑→小实体（跳空）→长白（切入长黑）。底部反转。
- **弃婴形态**：十字星与前后都有缺口，极罕见，信号极强。

#### 窗口（缺口）
- 上升窗口=支撑，下降窗口=阻力
- 收盘价封住窗口=趋势结束
- 三窗口=极度超买/超卖，物极必反
- **反转信号+同向窗口=双倍确认**

### 蜡烛图五大原则
1. **先确定趋势** — 上升趋势中只做多（多头信号买入，空头信号平多），下降趋势反之
2. **信号≠交易** — 必须计算风险收益比，不划算不做
3. **止损必须设** — 形态关键位下方/上方
4. **做市场变色龙** — 情况变了就认，不死撑
5. **形态位置比形态本身更重要** — 支撑位上的刺穿 >> 随机位置的空头吞噬

### 艾略特波浪理论（Murphy Ch12）

#### 核心：5-3节奏
- **驱动浪（5浪）**：1启动→2回调（不破1起点）→3主升（最长最猛）→4整理（不进入1浪范围）→5末升
- **调整浪（3浪）**：A下跌→B反弹（逃命机会）→C主跌

#### 铁律
1. 2浪不破1浪起点
2. 3浪永远不是最短的（在1/3/5中）
3. 4浪不进入1浪价格范围（股市严格，期货可容忍日内穿越）

#### 各浪特性与交易含义
| 浪 | 特点 | 交易策略 |
|----|------|---------|
| 1浪 | 底部启动，悲观中诞生 | 轻仓试探 |
| 3浪 | 最猛、最长、放量 | **重仓持有** |
| 5浪 | 量萎缩，指标背离 | 减仓，准备离场 |
| B浪 | 量轻，反弹 | 最后的逃命机会 |

#### 菲波纳奇比数
- 3浪目标 = 1浪×1.618 + 2浪底
- 5浪目标 = 1浪×3.236
- C浪目标 = A浪×1.618
- 回撤位：38.2%、50%、61.8%
- 时间目标：13、21、34、55、89根K线

#### 调整浪类型
- **锯齿形(5-3-5)**：凶猛下跌
- **平台形(3-3-5)**：温和横盘
- **三角形(3-3-3-3-3)**：收敛整理
- **交替规则**：2浪和4浪形态通常不同

#### 波浪理论使用原则
- 只在浪型清晰时使用
- 必须与其他工具（趋势线、均线、指标）交叉验证
- **不要强行数浪** — 看不出来就不数
- 配合K线反转信号确认转折点

## L3：交易系统 —— 系统化捕捉趋势

### 趋势跟踪的本质（Clenow）

**核心哲学**：价格沿已形成方向继续运行。不预测，只跟随。

只需要做三件事：
1. **等** — 等到明确的趋势出现
2. **跟** — 押注价格沿同方向继续运行
3. **退** — 趋势反转时退出

**关键认知**：胜率只有30%-42%，但少数大赢远远覆盖大量小亏。收益分布呈厚尾右偏。趋势跟踪是反人性的——高位买、低位卖、大多数交易亏钱、盈利反复回吐。

### 五大成功因素（Clenow — 远比具体规则重要）

1. **分散化**：覆盖5个板块（股票、利率、外汇、农产品、非农），每板块≥10品种，总≥50
2. **持仓限额**：ATR仓位公式，让每个品种对组合的日波动贡献相等
3. **时间尺度**：选择跟踪多长周期的趋势（中期=6-7周持仓）
4. **风险水平**：风险因子0.1%-0.25%，调节风险和收益的唯一旋钮
5. **多策略**：同一策略用于不同时间尺度，平滑业绩

### Clenow核心策略公式

```
开多条件：50日均线 > 100日均线 AND 收盘价 = 过去50天最高收盘价
开空条件：50日均线 < 100日均线 AND 收盘价 = 过去50天最低收盘价
多头止损：开仓以来最高收盘价 - 3 × ATR
空头止损：开仓以来最低收盘价 + 3 × ATR
合约数量 = floor(账户总值 × 风险因子 / (ATR × 点价))
```

### 海龟交易系统（Faith — 完整规则）

**头寸单位**：`单位 = 账户净值×1% / (N × 每点合约价值)`
- N = 20日ATR
- 1单位 = 使单日波动约等于账户1%的合约数

**入市信号**：
- 系统1（短线）：20日高点突破做多，20日低点突破做空
- 系统2（长线）：55日高点突破做多，55日低点突破做空

**建仓规则**：每向有利方向移动1/2N加1单位，最多4单位。不一次建满。

**止损规则**：初始止损2N（距最新单位）。加仓后整体止损始终保持在距最新单位2N处。

**退出规则**：
- 系统1：10日反向突破退出
- 系统2：20日反向突破退出

**仓位限制**（防毁灭性风险）：
- 单一市场≤4单位
- 高度相关市场同向≤6单位
- 单一方向总计≤12单位

**战术要点**：限价单优于市价单；买强卖弱（在信号品种中挑最强的买，最弱的卖）；到期前滚动合约。

### 两种系统的对比与选择

| | Clenow系统 | 海龟系统 |
|---|---|---|
| 时间框架 | 50日突破 | 20日(S1)或55日(S2)突破 |
| 过滤器 | 50日>100日均线 | 25日 vs 350日均线 |
| 止损 | 3ATR | 2N(≈2ATR) |
| 仓位 | ATR自适应 | N值头寸单位 |
| 退出 | ATR移动止损 | 反向突破 |

### 利弗莫尔的交易体系（Livermore）

**完整流程**：
1. 大势研判 — 牛市还是熊市？什么阶段？
2. 板块选择 — 做多选最强板块，做空选最弱板块
3. 个股选择 — 选领头羊，成交活跃
4. 时机等待 — 等市场自己给出信号，等关键点突破
5. 分批建仓 — 试探→确认→加仓→再确认→再加仓（金字塔式）
6. 坚定持仓 — 忽略小波动，关注大趋势是否改变
7. 果断离场 — 趋势明确反转时，不犹豫

**关键点方法**：整数关口（100, 200, 300）、历史新高/新低。突破即可入场。

**核心规则**：
- 永远不摊平亏损头寸
- 盈利头寸保留，亏损头寸砍掉
- 没有把握就等，不要天天交易
- 不听信内幕消息，独立判断

## L4：风险管理 —— 活下来才能赢

### 三大黄金法则（Murphy Ch15 + 全体系共识）

1. **顺势而为** — 主要趋势向上只做多；主要趋势向下只做空
2. **让利润奔跑** — 移动止损跟随趋势，不给止盈给止损
3. **赔小赚大** — 单笔亏损可控，赚钱的单子让利润充分增长

### 仓位管理

- 单笔交易风险不超过总资金2%（保守1%甚至0.5%）
- 初次入场用30%-50%资金，确认后逐步加仓
- 关联品种视为同一仓位
- 亏损头寸绝不平摊

### 止损设置

| 方法 | 来源 | 适用 |
|------|------|------|
| 动态跟踪止损（ATR倍数） | Clenow, Faith | 趋势跟踪的标配 |
| 形态外侧止损 | Murphy | 突破交易 |
| 固定百分比止损 | Murphy | 所有交易 |
| 支撑/阻力位外侧 | Livermore, Murphy | 通用 |

### 盈亏比

进场前先算账：预期盈利 / 预期亏损 ≥ 2:1。不到不做。

### 资金管理核心公式

```
海龟头寸：单位 = 账户净值 × 1% / (N × 每点价值)
Clenow仓位：合约数 = 账户总值 × 风险因子 / (ATR × 点价)
单笔最大风险：≤ 账户总额 × 2%
```

### 流动性风险（Livermore + Faith）

大额头寸的买卖行为本身会影响价格。出货需要活跃市场环境，通常在上涨过程中完成。头寸规模限制的真正作用是防止多个高度相关市场同时剧烈反向时的叠加损失。

### 生存第一法则（Faith）

交易的首要目标是生存。两种死亡形式：
- 慢性死亡：连续亏损导致自我怀疑、彻底放弃
- 快速死亡：一次重创直接倾家荡产

时间站在你这边。只要留在游戏场中，正期望值的系统早晚让你赚钱。

## L5：心理修炼 —— 战胜自己

### 概率思维（Douglas核心）

**不接受预测，只接受概率**。你无法知道下一笔交易赚还是赔，但你知道：
- 你的系统在足够多样本下有正期望值
- 亏损是概率的正常体现，不代表你做错
- 赚钱的交易不代表你做对（可能是运气）
- 像赌场一样操作：不挑机会，按系统执行每一个信号

### 四种基本交易恐惧（Douglas）

1. **害怕犯错** → 错过信号、不敢执行
2. **害怕亏钱** → 不止损、不让利润奔跑
3. **害怕错过** → 冲动进场、违反规则
4. **害怕失算** → 提前平仓、不敢持有

**最优秀的交易者不害怕。** 他们根据市场机会进出，形成了最大的思想灵活性。

### 七项持续一致性原则（Douglas）

1. 客观确认优势
2. **提前衡量风险**（进场前就设好止损）
3. 完全接受风险（接受这笔可能亏损）
4. 毫不犹豫地行动
5. 兑现利润（付钱给自己）
6. 自我监控
7. 永不违反

### 8种认知偏差（Faith — 利用别人的，克服自己的）

| 偏差 | 他人如何被利用 | 自己如何克服 |
|------|---------------|-------------|
| 损失厌恶 | 恐慌抛售→低价买到筹码 | 用系统化止损强迫执行 |
| 沉没成本 | 死扛亏损→趋势延续 | 每笔交易独立看待 |
| 处置效应 | 赚一点就跑→趋势延续 | 截断亏损，让利润奔跑 |
| 结果偏好 | — | 不根据单笔结果评判决策 |
| 近期偏好 | — | 看20年数据，不只看最近 |
| 锚定效应 | 在前期高低点反弹→趋势延续 | 突破时跟进 |
| 潮流效应 | 从众追涨杀跌→极端价格 | 独立思考，不盲从 |
| 小数定律 | — | 用大样本回测，不轻易下结论 |

### 利弗莫尔的心理智慧

- **投机者的主要敌人藏在内部** — 自己的希望、恐惧、贪婪、自负
- **大钱不是从个别波动中来的，而是从主要趋势中来的**
- **挣大钱凭"无为"，不凭"作为"** — 耐心坚守
- **不要试图抓住最后八分之一美元** — 追求极致精确的代价太高
- **看对与做对是两回事** — 在等待市场验证中保持不动摇才是真功夫
- **破产是最好的老师** — 每次失败都要找出原因、修正体系
- **市场永远没错，错的只能是你**

### 20笔交易训练法（Douglas）

1. 选一个精确的交易系统，测试赢亏比
2. 承诺交易连续的20个信号 — 不挑、不跳
3. 不能偏离系统任何参数
4. 不在完成20笔前改变任何东西
5. 目标是体验概率在宏观层面兑现

如果内心有冲突，训练会非常困难 — 困难程度=内在冲突程度。完成时无任何阻力=概率思维已内化。

### 建立信心的正确方法（Faith）

1. **亲手用历史数据检验你的系统** — 别人说有效没用
2. **研究交易记录，摸清亏损的频率和程度** — 知道过去有8个月的亏损期，现在的8个月就不怕
3. **理解系统每个环节为什么有效** — 知道快速建仓是系统关键，就不会"为了降低风险"放慢速度

## 实用工作流：从分析到执行的完整步骤

### 第一步：大势研判（多时间框架）
1. 看周线图/月线图：主要趋势方向？200日均线方向？
2. 看日线图：次级趋势如何？当前处于哪个波浪位置？
3. ADX > 25 → 有趋势可追；ADX < 20 → 震荡市，用摆动指标

### 第二步：支撑阻力识别
1. 画主要趋势线
2. 标出前期高、低点
3. 标出菲波纳奇回撤位（38%, 50%, 62%）
4. 标注整数关口

### 第三步：形态识别
1. 有价格形态在形成吗？反转还是持续？
2. K线是否给出反转信号？（支撑位上的锤子线 > 随机位置的锤子线）
3. 交易量是否验证？

### 第四步：信号确认（多重验证）
- 趋势工具（趋势线/均线） + K线信号 + 摆动指数（背离/极端） + 交易量 → 四重确认最强
- **至少两个工具给出同向信号才出手**

### 第五步：制定交易计划
- 入场点？止损点？第一目标？第二目标？
- 盈亏比 ≥ 2:1？
- 仓位大小？分几次建仓？

### 第六步：执行与管理
- 到了入场点就执行，不犹豫
- 趋势正确 → 移动止损跟随
- 出现反转信号 → 果断离场
- **唯一目标：按计划执行，不评判单笔结果**

## 数据驱动分析流水线（binpan 集成）

通过 [binpan](https://pypi.org/project/binpan/)（Binance API 封装库），本 skill 可直接获取实时行情、计算指标、扫描信号和回测策略。详见 [references/binpan-integration.md](references/binpan-integration.md)。

### 六步分析流水线

**Step 1 — 数据准备**：`python scripts/fetch_klines.py SYMBOL INTERVAL LIMIT -o data.csv` 然后 `python scripts/compute_indicators.py data.csv -o indicators.csv`
**Step 2 — 信号扫描**：`python scripts/detect_setup.py indicators.csv` 输出所有检测到的 K线形态、突破信号、指标信号，每个信号标注来源书系
**Step 3 — 交易计划**：对每个信号确认趋势方向 → 计算止损/目标/仓位 → 盈亏比 ≥ 2:1 才执行
**Step 4 — 回测验证**：`python scripts/backtest_system.py indicators.csv --strategy both` 输出 CAGR/Max DD/MAR/Sharpe/胜率/盈亏比/收益偏度
**Step 5 — 组合分析**：`python scripts/analyze_portfolio.py --symbols ... --account N` 多品种 ATR 仓位计算 + 相关性检测
**Step 6 — 图表输出**：`python scripts/plot_analysis.py indicators.csv -o chart.html` 生成带指标、S/R、一目均衡表的交互式图表
